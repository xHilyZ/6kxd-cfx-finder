# 6kxd CFX Finder

A fast, clean FiveM server lookup tool with real‑time stats, auto‑refresh, player/resource explorer, and a modern glass UI.

## Features
- Advanced CFX auto‑detect (codes, links, messages)
- Live server status (Online / Slow / Offline)
- Player & resource counts
- Server banner + icon
- GeoIP country detection
- Ping heatmap
- Auto‑refresh (10s / 30s / 60s)
- Full player & resource panel with search + sorting
- JSON viewer (players + server info)
- Recent lookups

## Tech
- HTML / CSS (Glass UI)
- Vanilla JavaScript
- CFX.re API
- ipapi.co GeoIP
