## Features
- Auto‑detect CFX codes from any text
- Live server status (Online / Slow / Offline)
- Player + resource counts
- Server banner + icon
- GeoIP country lookup
- Ping heatmap
- Auto‑refresh (10s / 30s / 60s)
- Full player/resource panel with search + sorting
- JSON viewer
- Recent lookups

## Tech
HTML • CSS • JavaScript • CFX.re API • GeoIP API
